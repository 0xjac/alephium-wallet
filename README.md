# Alephium Wallet

The official Alephium wallet.

## Install

```shell
npm install
```

## Packaging

### Web extension

```shell
npm run extension-pack
```

### Desktop app

```shell
npm run electron-pack
```

## Development

```shell
npm start
```

## Test

```shell
npm test
```
