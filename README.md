# Alephium Wallet

The official Alephium wallet.

## Install

```shell
npm install
```

## Packaging

### Web extension

```shell
npm run extension-pack
```

### Desktop app

```shell
npm run electron-pack
```

## Development

To launch the app as a web app, run:

```shell
npm start
```

To launch it as an electron app, run:

```shell
npm run electron-dev
```

## Test

```shell
npm test
```
